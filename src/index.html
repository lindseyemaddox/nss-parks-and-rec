Parks and Recreation

Part One

Build an application using grunt to get data from the following API: https://raw.githubusercontent.com/nss-day-cohort-31/national-parks/master/database.json.

  1. Display the list of national parks on the DOM.
  2. Each national park should be formatted in HTML like below:
    3. <article>
        <h3>Park Name</h3>
        <p>State the park in located in</p>
    </article>
  3. If the park has been visited, the article tag should have a red dashed border. If the park has not been visited, it should have a green solid border.
