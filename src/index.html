Parks and Recreation

Part One

Build an application using grunt to get data from the following API: https://raw.githubusercontent.com/nss-day-cohort-31/national-parks/master/database.json.

  1. Display the list of national parks on the DOM.
  2. Each national park should be formatted in HTML like below:
    3. <article>
        <h3>Park Name</h3>
        <p>State the park in located in</p>
    </article>
  3. If the park has been visited, the article tag should have a red dashed border. If the park has not been visited, it should have a green solid border.

Part Two

For each of the national parks, use the latitude and longitude to get the weather. The Dark Sky API allows you to request weather forecasts and historical weather data programmatically. It is the easiest, most advanced weather API on the web.

  1. From the data coming back from the Dark Sky API, you are going to use summary listed under currently, hourly and daily.
  2. Change your original html to look like this:
    3. <article>
        <h3>Park Name</h3>
        <p>State the park in located in</p>
        <p>Weather:</p>
        <ul>
            <li>Currently: Summary from currently in API Data</li>
            <li>Today: Summary from hourly in API Data</li>
            <li>Week: Summary from daily in API Data</li>
        </ul>
    </article>
